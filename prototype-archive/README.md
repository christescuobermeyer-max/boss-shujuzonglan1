# 代运营数据统计系统 Design System

## 公司 / 产品概述

**公司性质：** 餐饮代运营服务商，专注于美团、饿了么平台的线上代运营业务。与商家签订线上代扣合同，按每笔开单金额的 **8%** 收取服务费。

**业务范围：**
- 全国各地商家开单管理（订单来自全国多个省份）
- 两大销售部门：**武汉部** 和 **宜昌部**
- 运营团队负责接管店铺的日常运营

**数据统计系统核心功能：**
1. **中国地图** — 各省份开单总数热力展示
2. **销售开单数** — 每个销售人员的开单数量统计
3. **运营接单** — 每个运营人员管理的店铺数量
4. **武汉 vs 宜昌** — 两个销售部开单对比
5. **每日抽点 & 回款** — 总抽点店铺数 / 总回款店铺数
6. **回款金额** — 按城市（武汉/宜昌）/ 运营人员 / 平台（美团/饿了么）多维拆分

**数据来源：** 云数据库（网页通过 API 读取）

## CONTENT FUNDAMENTALS

**语言：** 简体中文为主，数据单位用英文（%、￥、个、家）

**文案风格：**
- 简洁直接，数字突出，标签简短（如"开单数"而非"开单总数量"）
- 无 emoji，专业商务风格
- 数字使用千分位格式：`12,345`
- 金额格式：`￥12,345.00`
- 日期格式：`2024-01-15`

**语气：** 中性客观，数据说话，无营销语言

---

## VISUAL FOUNDATIONS

**主题：** 浅色（Light Theme）

**配色：**
- 背景：`#F5F7FA`（页面底色）、`#FFFFFF`（卡片）、`#EEF2F8`（次级背景）
- 主色：`#1677FF`（品牌蓝，操作/高亮）
- 辅助色：`#13C2C2`（青色，饿了么/对比色）
- 武汉色：`#FF6B35`（橙红）
- 宜昌色：`#1677FF`（品牌蓝）
- 美团色：`#FFD100`（美团黄）
- 饿了么色：`#0099FF`（饿了么蓝）
- 成功/正向：`#52C41A`
- 警告：`#FAAD14`
- 文字主：`#1D2129`
- 文字次：`#4E5969`
- 文字辅：`#86909C`
- 边框：`#E5E8EF`

**字体：** Noto Sans SC（中文）+ Inter（数字/英文）
- 数据大字：48–64px，Bold
- 卡片标题：16px，Medium
- 正文标签：14px，Regular
- 辅助文字：12px，Regular

**布局：**
- 最大宽度：1920px，固定宽度大屏
- 栅格：24列，间距16px
- 圆角：卡片 `8px`，按钮 `6px`，标签 `4px`
- 卡片：白色背景，`0 2px 8px rgba(0,0,0,0.08)` 阴影，1px `#E5E8EF` 边框

**动效：** 轻微，`ease-out 0.2s`；数字滚动动效；hover 时卡片阴影加深

**图表：** ECharts，浅色主题；折线/柱状使用品牌蓝渐变；地图使用蓝色热力

## ICONOGRAPHY

无专属图标库，使用以下方案：
- **数字/图表图标：** 内联 SVG
- **平台图标：** 美团（黄色）、饿了么（蓝色）用品牌色色块区分
- 无 emoji，无外部图标字体
